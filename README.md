# 🐾 Animal Shelter Adoption Analysis (Power BI)

## 📌 Project Overview
This project presents an analysis of animal shelter data (dogs and cats) with a focus on adoption trends and shelter capacity.

The goal was to identify key patterns in adoption rates and highlight differences between animal groups to support better decision-making for animal welfare organizations.

---

## 🎯 Business Problem
Animal shelters need to understand:
- how adoption rates change over time,
- whether the number of animals in shelters is increasing or decreasing,
- which animal groups require more support.

This dashboard aims to provide clear insights to support adoption strategies and resource planning.

---

## 📊 Key Insights

- 📉 Since 2020, the number of dogs in shelters has been decreasing, along with dog adoptions.
- 📈 At the same time, the number of cats in shelters and cat adoptions has been increasing.
- 📊 Dogs account for approximately **79% of all adoptions**.
- 📉 The adoption rate peaked around 2020 and has slightly declined in recent years.
- 🏠 The number of shelters has been increasing over time, with slight stabilization in recent years.

---

## 📈 Dashboard Features

- Adoption rate analysis over time
- Comparison of dogs vs cats (adoptions & shelter population)
- KPI indicators:
  - Adoption rate
  - Total number of adoptions
- Trend analysis (yearly changes)
- Clear storytelling using annotations and highlights
